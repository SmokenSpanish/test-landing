# Test Landing Page

Это тестовое задание по верстке страницы сайта. Проект реализован на HTML, SCSS.  

## Особенности проекта

Адаптивная ("резиновая") верстка для десктопа и мобильных устройств

## Установка и запуск
```bash
npm install
Для разработки с автоматической пересборкой CSS и локальным сервером:
npm run dev

CSS будет пересобираться при изменениях в scss/main.scss

Локальный сервер будет доступен по адресу http://127.0.0.1:8080 (по умолчанию)

[Посмотреть сайт онлайн](https://smokenspanish.github.io/test-landing/)
